Project

36 Rep Max Calculator

This was my first project which included HTML, CSS with Bootstrap, and Vanilla Javascript. I often used a 20rep max calulator during or outside of my strength training workouts to figure out my 1 rep max each time to plan my appropriate sequential weights and reps for the next workout. This is the first website that I know of that calculates both bodyweight and external weight with reps to calculate one rep max as well as extend past 20 reps.


Note: This was the first website I've ever hosted. I use it at least a few days a week and has proven to be very useful.

NFT Project Template

I was a part of an NFT project. I downloaded the entire Repo from Hashlips via Youtube to have access to the codes to accept Ethereum payments. We had a web designer on our team and therefore I had to translate his designs from Dreamweaver and recreate it using React since that was the language. This was my first time translating design from a designer as well as developing the visuals using React. I also analyzed the code enough where I was able to create a ternary oprator that changes the text of the wallet button(top right) wether the user was signed in. The languages I had to learn for this project was HTML, CSS, React, and Solidity (to understand and execute smart contracts that enable successful ETH transactions.)
